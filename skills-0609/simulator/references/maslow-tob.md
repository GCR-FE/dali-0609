# 马斯洛需求层 · 在 ToB 销售场景中的应用

> **用途**:马斯洛回答"他现在需要什么"—— 是六框架级联中的第三层,穿透到**当前被激活的需求**。
>
> **关键**:这是**动态层**,随环境变化。跟九型(恒定动机)不同 — 同一个九型 8 号的人,在不同处境下被激活的马斯洛层会不同。

## 五层需求在 ToB 场景的翻译

| 层 | 原始定义 | ToB 场景翻译 | 典型激活场景 |
|---|---|---|---|
| **5 · 自我实现** | 成为最好的自己 | 做出被业界记住的事;留下专业遗产 | 职业后期 / 二次创业 / 转型领袖 |
| **4 · 尊重** | 被认可、自尊 | 被上级认可;在董事会有话语权;行业地位 | 上升期管理者 / 竞争白热化时 |
| **3 · 社交** | 归属、爱、友情 | 团队归属;专业社群;跟同级的关系 | 新上任 / 跨部门合作期 |
| **2 · 安全** | 安全、稳定 | 岗位稳定;不被裁;财务健康;合规 | 组织变革 / 业绩下滑 / 行业震荡 |
| **1 · 生理** | 生存基础 | 短期现金流;项目活下去 | 严重危机期(少见于 ToB 决策层) |

## 激活逻辑 · 环境如何决定哪层被激活

马斯洛的关键点:**当下被激活的是哪一层,就决定了他当下怎么做决策**。即使平时这个人追求自我实现(5 层),一旦遇到裁员威胁,会瞬间降到安全层(2 层)。

**常见激活模式**:

| 外部环境 | 典型激活层 | 行为转变 |
|---|---|---|
| 公司业绩下滑 / 预算压缩 | 安全层(2) | 过去谈愿景的人突然只问 ROI |
| 组织变革 / 新领导上任 | 安全层(2)或尊重层(4) | 过去果断的人变得谨慎观望 |
| 行业震荡 / 竞争恶化 | 安全层(2) | 过去开放的人变得保守 |
| 新业务成功 / 获得认可 | 尊重层(4)或自我实现层(5) | 开始谈长期愿景和行业影响 |
| 跨部门合作期 | 社交层(3) | 开始关注人际关系、团队氛围 |
| 职业后期 / 临近退休 | 自我实现层(5) | 更愿意做长远但有意义的事 |

## 如何判断当前被激活的层

### 直接信号

| 观察到 | 可能被激活的层 |
|---|---|
| 反复问"最坏情况""有没有保底" | 安全层(2) |
| 在意"董事会会怎么看""同行会怎么说" | 尊重层(4) |
| 主动聊团队凝聚力 / 新人融入 | 社交层(3) |
| 谈"我希望这个行业变成 X 样""留下 Y" | 自我实现层(5) |
| 在意"合规""不出事" | 安全层(2) |
| 展示成就、奖项、个人影响力 | 尊重层(4) |

### 间接信号 · 环境调查

**Phase 0 的结构化提问应覆盖**:
- 他所在公司最近的业绩 / 战略变化?
- 他个人的职业阶段(新上任 / 稳定期 / 临近退休)?
- 这个项目对他个人的意义?
- 他的上级 / 下级近期有什么变动?

> **销售场景应用**:环境调查的答案直接决定你怎么跟他谈话。同一个 CFO,公司扩张期跟公司预算收缩期,能打动他的完全是不同的东西。

## 层级转换的销售应对

| 他在哪层 | 销售怎么对话 |
|---|---|
| **安全层(2)** | 强调"不出错""有保底""可退出";用"保守型""已验证"的包装;多给风险缓冲机制 |
| **社交层(3)** | 关注团队如何受益;讲"我们怎么一起"而不是"你怎么做";关系经营优先 |
| **尊重层(4)** | 帮他在组织中赢面子;准备给他用的对上/对外汇报材料;强调"首创""领先""被同行看到" |
| **自我实现层(5)** | 谈长期影响、行业意义、个人遗产;讲愿景;他不在乎短期 ROI |

## 马斯洛与其他框架的协作

- **马斯洛看当前,九型看恒定**:同一个九型 3 号(恐惧失败),在安全层被激活时表现谨慎保守,在尊重层被激活时表现激进争胜 — 两种状态都是他,但应对方式完全不同
- **马斯洛看外部激活,DiSC 看行为风格**:外部压力让 D 型临时变 S 型,不能把临时变化当成 DiSC 的重定位(参见 `stress-detection.md`)
- **马斯洛是销售策略的关键决定因素**:画像里如果不识别当前激活层,销售动作很容易错配

## 不要做的判定

- ❌ 把马斯洛当恒定型来判定 — 它是动态的
- ❌ 忽略环境调查直接判断当前层
- ❌ 对 User 说"他现在处在安全层" — 永远翻译成"他现在最在意的是 X"
- ❌ 用单次观察判定被激活层 — 需要跨场景一致性
