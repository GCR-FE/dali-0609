## Asset 详细要点

### 1. MDI — 医学深入研究智能体平台

- **场景**：新药研发文献检索、数据整理、报告撰写
- **关键 KPI**：
  - 检索时间：180-300 分钟 → 2-3 分钟（缩短 99%）
  - 人力：5 人团队 → 2 人团队（减少 60%）
  - 项目产能：2-3 个/年 → 6-8 个/年（翻倍）
  - 年度成本：87 万元 → 28.5 万元（节省 58.5 万元）
- **核心技术**：TxGemma 医学模型 + 并行检索 + 知识图谱

### 2. MIH — 智能医学内容生成中心（**复星医药已生产**）

- **场景**：临床试验报告、IND/NDA 监管文件、医学翻译
- **关键 KPI**：
  - 报告一致性检查：1 周 → 5 分钟（缩短 99.85%）
  - 翻译量：6 个月完成 1.6 亿字
  - 90% 撰写场景覆盖
  - 年营收 $500M 制药企业可获得 $23M 直接经济价值（$15M 营收 + $8M 成本）
- **公开案例**：复星医药（生产环境）、罗氏（MIH 翻译）

### 3. PC-UI — 图形化 HPC 药物研发平台

- **场景**：HPC 集群（虚拟筛选、分子动力学、AlphaFold）
- **关键 KPI**：
  - 集群创建：2-4 小时 → 10 分钟内
  - 试验运行速度：10 倍提升（弹性并行）
  - TCO 降低 30-50%（消除前期硬件投资 + 运维 + Spot 实例 60-90% 折扣）
  - 资源利用率：30% → 80%+

### 4. PFW — 蛋白质结构预测工作台

- **场景**：替代冷冻电镜，AI 预测蛋白质结构
- **关键 KPI**：
  - 单蛋白质结构解析：$50-100K → $5-10K（节省 80-90%）
  - 时间：数月 → 数小时
  - AlphaFold CASP14 GDT 92.4/100（接近实验室精度）
  - 任务分离架构（CPU MSA + GPU 预测）节省 50%+ 计算成本

### 5. AI4S — 研发智能体平台

- **场景**：多项目并行 AI 筛选 + MLOps
- **关键 KPI**：
  - 研发周期：12 个月 → 8 周（缩短 75%）
  - 单项目筛选成本：$2-3M → $500-800K（节省 60-70%）
  - GPU/CPU 按需付费节省 60-70% 基础设施成本
  - MLOps 成熟度：月级 → 周级

### 6. MCS — 医药营销商业化套件

- **场景**：医药代表培训、客户拜访准备、知识管理
- **关键 KPI**：
  - 销售成功率提升 20-30%
  - 新人上岗时间缩短 50%
  - 案例搜索：2-3 小时 → 5 分钟
  - 知识复用率：20% → 80%
- **公开案例**：拜耳（生产）、Takeda（生产，凯捷交付）
