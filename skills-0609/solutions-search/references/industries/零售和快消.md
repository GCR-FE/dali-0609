## Asset 详细要点

### 1. MCG — 智能营销内容生成

- **场景**：电商/快消的批量营销素材
- **关键数字**：
  - 单素材制作周期：3 天 → 0.5 天（缩短 83%）
  - 单人产能：9.5 小时/件 → 2.1 小时/件（提升 78%）
  - 真实案例：月产 28000 张素材，6 人组减少 2 人，节省 ¥24 万/年
- **品牌一致性**：通过设计调整智能体提示词保障批量素材统一

### 2. EPS — 智能电商选品

- **场景**：多 Agent 协同分析市场趋势 / 评论情感 / 竞品价格 / 历史销售
- **关键数字**：
  - 选品成功率：30% → 65%（基于行业数据 55-70% 取中位）
  - 单品类选品决策：16-24 小时 → 2 小时
  - 单品类调研周期：23 天 → 2 天（节省 21 天）
- **成本优化**：人力 -60%、试错损失 -80%，综合 -70%

### 3. Smart Shopping Guide — 智能导购（Rufus 模式）

- **场景**：基于情景推荐 / SKU 对比 / 模糊推荐
- **典型查询**："油性肌肤适合什么产品"、"给老婆买生日礼物香水"
- **关键数字**：
  - 转化率提升 20-30%（取 25%）
  - 用户满意度 +40%（7×24 小时 + 多轮对话）
  - 客服成本 -50%（AI 处理标准化咨询）
- **架构**：Agent 闭环 — 查询理解 → 证据检索 → 内容生成

### 4. CSA — 电商售后智能体

- **关键数字**（最猛的一个）：
  - 单客服日处理量：16 → 160 个问题（**+900%**）
  - 处理时间：30 分钟 → 3 分钟
  - 自助解决率：80%
  - 年客服运营成本：$1M → $550K（-45%）
  - 复购率 +12%
- **次级价值**：自动生成售后问题分析报告 → 产品迭代周期从 6 个月 → 2 个月

### 5. HRI — 酒旅商机洞察（GraphRAG）

- **场景**：从社交媒体热榜 / 赛事网站 / 周边热点信息构建知识图谱，给酒店定价、获客、营销
- **关键数字**：
  - 商机识别准确率：60% → 85%（GraphRAG 多跳推理）
  - 营销精准度 +35%（转化率 20% → 27%）
  - 运营成本：¥100 万 → ¥75 万（-25%）
- **技术验证依据**：GraphRAG 比传统 RAG +20-30%、Stanford KG-RAG +15-25%、AWS Neptune+Bedrock 综合 82-87%

### 6. ECM — 电商站点迁移（**资产已不再支持**）

- 但底层方案数据仍可用：
  - SLA 99.99%（vs 黑五前宕机 1 天）
  - 基础设施成本 -51%（AWS 官方数据）
  - 切换停机控制在数小时

### 7. ChatBI — 旅游酒店 BI（**德比软件 公开**）

- **公司**：德比软件（197 国家 / 22.7 万家酒店 / 1800 万间夜/月）
- **关键数字**：
  - SQL 生成准确率 **90%**
  - 50% 定制化 BI 需求由 ChatBI 自动完成
  - 业务查询开发效率 +50%
- **AWS 服务**：Bedrock、OpenSearch、Redshift、RDS、DynamoDB

## 公开赢单案例：Sugargoo 跨境电商迁移

**客户**：Sugargoo（2020 年成立，跨境电商，海外个人 + 中小企业）

**痛点**：
- 业务流程复杂（采购/质检/仓储/物流），运营成本高
- 原系统响应慢、API 延迟、客户体验差
- 信息孤岛、缺乏精准营销
- 物流配送低效、缺路线优化
- 安全合规风险

**业务成果（5 个月项目周期）**：
- 订单量 +20%
- 运营人员从 100+ → 40+
- 仓库管理从 200+ → 110
- 营销：网红 ROI 量化 + 广告投放渠道
- 采购：商品图片识别 + 爬虫比价 → 自动化
- 质检/履约：十几个逆向流程优化
- 物流：最优运输路线匹配，降本提速

**赢单 3 大要素**：
1. 深挖客户行业需求和痛点 → AWS 业务团队基于实际痛点定制方案
2. 整合行业解决方案内外部资源 → Account 团队 + ISVT + 合作伙伴**海邦亿腾**分工
3. 定制电商迁移方案 → 端到端用户旅程（埋点 / 行为分析 / 智能搜推）

**合作伙伴：海邦亿腾** — 出海电商系统定制化解决方案专家，提供代码开发 + 实施 + 业务改造。

**目标客户画像**：阿里云上的中型/大型出海电商客户。
