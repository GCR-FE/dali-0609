## 已知合作伙伴产品

### Cloud Lab（云加教学实验和科研云平台）

- **服务商**：Union Plus（北京）/ 西云数据
- **平台**：Cloud Plus Teaching Experiment and Research Cloud Platform
- **目标**：满足"新工科"建设需求
- **场景**：新医研、教育行业、大型企业
- **状态**：⚠️ This version is not available to new customers（已停售）

### ModelWhale（和鲸数据科学协作平台）

- **服务商**：Heywhale 和鲸科技 / 西云数据
- **平台**：数据科学协作平台
  - 支持 Python + R
  - Jupyter Notebook 交互 + Canvas 拖拽分析
  - 数据管理、建模分析、模型训练管理、计算资源管理、任务管理
- **规模**：30,000+ 用户，400,000+ 科研分析
- **典型客户**：
  - **科研**：中国气象局国家气象信息中心、中国医学科学院医学信息研究所、清华、南开、华东师大
  - **教学**：北大、上交、北京协和、同济、中国石油大学
  - **政企**：中国电信、中国银联、北京医保局、默克（Merck）中国
- **状态**：⚠️ This version has been removed and is no longer available to new customers（已下架）

## 涉及 AWS 服务（基于资料推断）

- **科研计算**：AWS Batch（Research Lab）、SageMaker、HPC
- **数据**：S3（科研数据集）、Open Data
- **教学应用**：EC2（Moodle on AWS）、CloudFormation
- **联络/触达**：Amazon Connect、SES、SMS、Pinpoint
- **AI/ML**：Bedrock、SageMaker（推断，资料无明确说明）

## 适用客户画像

- **985/211 高校**：清华、北大、上交、复旦、浙大、华东师大、南开、同济
- **专业医学/科研院校**：北京协和、中国医学科学院、中国气象局
- **职业院校**：以"职业院校专业建设""1+X 证书"思路切入
- **K12 在线教育**：（资料缺失，需另行调研）
- **教培/职业培训**：与 Skills Competition 竞赛运营结合

进入路径（推测）：
1. **科研院所** → Research Lab / ModelWhale 类产品
2. **高校教学** → Cloud Lab / Moodle 学习系统 / Virtual Classroom
3. **职业教育** → Cloud Raiser 实训平台 / 大赛合作
4. **国际化/海外校区** → 弹性学习 + 合规
